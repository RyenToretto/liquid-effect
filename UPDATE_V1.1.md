## 🔧 自定义配置

想要调整水珠效果？非常简单！

### 修改数量
```typescript
// src/components/WaterDropCanvas.vue
const TOTAL_DROPS = 15  // 改为你想要的数量
const cols = 5          // 列数
const rows = 3          // 行数
```

### 修改大小
```typescript
const size = 40 + Math.random() * 60
// 改为: const size = 最小值 + Math.random() * (最大值 - 最小值)
```

### 修改动画
```typescript
// 呼吸幅度
const breathOffset = Math.sin(drop.breathPhase) * 3  // 改为更大的数字

// 旋转速度
drop.mesh.rotation.z += 0.0005  // 改为更大的数字
```

详细配置请参考 [WATER_DROPS_FEATURES.md](./WATER_DROPS_FEATURES.md)
